# 北京高考议论文思路训练器

> 专为北京高考议论文备考设计的 AI 辅助训练工具，帮助考生在限时压力下完成审题、立论、选据、搭框架的完整构思流程。

**线上体验**：https://gaokaozuowen-trainer.pages.dev

---

## 产品背景

北京高考语文作文要求考生在有限时间内写出结构清晰、论据充分、立意深刻的议论文。大多数学生的问题不在于"不会写"，而在于：

- **审题不准**：抓不住题目核心，跑题或偏题
- **立论泛化**：论点不够聚焦，沦为口号
- **论据单薄**：只会用几个固定案例，缺乏层次
- **结构混乱**：开头结尾割裂，主体段落逻辑不清

本工具以 2021—2025 年北京卷真题为训练素材，通过 AI 即时诊断，帮助学生建立"看到题目 → 快速构思 → 拿到反馈 → 优化思路"的训练闭环。

---

## 核心功能

### 1. 真题训练
涵盖 2021—2025 年北京卷议论文真题，每道题配有：
- 完整原题材料
- 2 篇高质量范文（可展开查看全文）
- 范文论点、论据、结构拆解

### 2. 限时构思（8 分钟倒计时）
模拟真实考场压力，要求在 8 分钟内完成：
- 中心论点（一句话概括）
- 三条论据
- 论证结构选择（总分总 / 并列 / 递进 / 对比 / 古今对比）

### 3. AI 智能诊断
提交构思后，AI 即时给出：
- **综合评分**（满分 10 分）
- **论点评价**：是否切题、是否有深度
- **论据评价**：是否匹配题意、层次是否丰富
- **结构评价**：逻辑是否自洽
- **补充建议** + **推荐论据**

### 4. AI 生成大纲
基于诊断结果，自动生成：
- 完整作文大纲（开头 / 主体三段 / 结尾）
- 开头示范段（150 字）
- 结尾示范段（150 字）

### 5. AI 模拟题
一键生成北京卷风格的全新模拟题，自动避开已练过的题目，支持持续刷题。

## 技术架构

```
gaokao_trainer/
├── index.html              # 单文件前端，纯 HTML/CSS/JS，无框架依赖
├── server.py               # 本地开发服务器（Python 标准库）
└── functions/
    └── api/
        └── chat.js         # Cloudflare Pages Function，代理 Kimi API 调用
```

- **前端**：原生 JS + CSS 变量，支持深色模式，移动端响应式
- **AI**：Kimi API（moonshot-cn），模型 `kimi-k2.5`
- **部署**：Cloudflare Pages（全球 CDN，免费托管）
- **本地运行**：Python 3 内置 HTTP 服务器，无需安装依赖

## 本地运行

```bash

# 启动服务器
python3 server.py

# 浏览器打开
open http://localhost:8080
```

Kimi API Key 申请：https://platform.moonshot.cn

## 训练建议

1. **先独立构思**，不要急着提交——8 分钟倒计时是真实的考场节奏
2. **对照范文差距**，重点看论据的层次感和论点的精准度
3. **每次练习后**，把 AI 推荐的论据记下来，丰富自己的素材库
4. **用模拟题**反复练习审题，培养对北京卷命题方向的敏感度
